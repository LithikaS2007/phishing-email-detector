# Phishing Email Detection Model

This project detects phishing emails using machine learning.

## Features
- Classifies emails as phishing or safe
- Uses Naive Bayes algorithm
- Displays accuracy and confusion matrix

## Technologies
- Python
- Scikit-learn
- Pandas

## How to Run
1. Install libraries:
   pip install pandas scikit-learn

2. Run:
   python phishing_model.py

3. Enter email text to test

## Example
Input: "Click here to win money"
Output: phishing
